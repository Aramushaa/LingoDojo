## Redundancy control
- Avoid multiple variants of the same intent in the same A1 pack.
- If variants are useful, move them to A2 glue or store as examples.

## A1/A2/B1 intent boundaries
- A1: survival + immediate action, short forms.
- A2: explanations, apologies, rebooking, announcements.
- B1: pressure, negotiation, disputes, authority language.
- Dark Mode: recognition only, risk + safer alternative.

## Dark Mode safety rule
Every Dark Mode card MUST include:
- `risk = avoid`
- `trap` (why it can escalate)
- `native_sauce` (safer alternative)
- `cultural_note` (tone + context)

## Product UI guidance
- Show packs by category (Tools / Mindset / Voice), then module, then level.
- Inside modules, list only pack title + level.
- Voice (dark / risky) requires a warning gate.
